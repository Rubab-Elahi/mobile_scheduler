AI Mobile Scheduler App
An intelligent Agentic AI-powered productivity app that helps users manage tasks, create smart schedules, and stay productive automatically.
Key Features
Smart Authentication
Using Amazon Cognito
* Secure Login & Signup
* Google Sign-In
* JWT Authentication
________________


AI Task Management
Users can:
* Add tasks manually or by voice
* Save tasks by date
* Set priorities
* Manage daily activities easily
All data is stored in Amazon DynamoDB.
